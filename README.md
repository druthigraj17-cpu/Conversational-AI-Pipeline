# Conversational-AI-Pipeline

## AI Engineering Assignment

This repository contains prototype implementations and workflow demonstrations for conversational AI systems, research-oriented LLM workflows, and autonomous browser AI agents using GPU-accelerated environments.

---

# Project Overview

The assignment focuses on building and understanding modern AI engineering pipelines involving:

* Speech-to-Text systems
* Conversational AI orchestration
* Local LLM workflows
* Research-oriented AI assistants
* Browser automation agents
* GPU-based AI inference

The implementations were developed and tested using Google Colab GPU environments as an alternative to RunPod deployment.
